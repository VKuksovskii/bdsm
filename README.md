# WP rehost

Bash script to easily
- export,
- import,
- search
- search/replace database of your website.

# Supported engines
- WordPress
- Laravel

## Installation:
```
. <(curl -L https://gogit.ru/wprehost)
```

## Usage

1. `cd` in website root directory
2. Execute `wprehost`
3. Follow instructions. Enjoy!
